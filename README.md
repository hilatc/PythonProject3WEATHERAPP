A Python web application built with Streamlit that allows users to check
current weather conditions and a 5-day forecast for any city in the world.

The app uses the OpenWeatherMap REST API and presents the data in a clear,
interactive, and user-friendly interface.

---

## Features
- Search weather by city name
- Current weather data:
  - Temperature
  - Feels-like temperature
  - Humidity
  - Wind speed
  - Weather description and icon
- 5-day weather forecast
- Interactive temperature graph (Plotly)
- Supports metric and imperial units
- Secure API key handling using Streamlit secrets

---

## Technologies Used
- Python
- Streamlit
- Requests (REST API calls)
- Pandas (data processing)
- Plotly (interactive graphs)
- OpenWeatherMap API
